üìñ How to Install

  1. Choose your device type above
  2. Click the "Install SNOLED" button
  3. Select your device's COM port when prompted
  4. Wait for the installation to complete (2-3 minutes)
  5. Connect to the SNOLED-AP WiFi network (password: wled1234)
  6. Configure your WiFi and LED settings

üîß After Installation

  • WiFi AP: SNOLED-AP (password: wled1234)
  • Web Interface: Connect to WiFi and visit http://snoled.local
  • Default Settings: No LEDs configured - set up in Config ‚Üí LED Preferences
  • Updates: Use the built-in OTA updater or this web installer

‚ùì Troubleshooting

  • Port not detected? Make sure you're using a data USB cable (not charge-only)
  • Flash failed? Try holding the BOOT button while connecting
  • Browser not supported? Use Chrome, Edge, or Opera (version 89+)
  • Device not responding? Power cycle your ESP device and try again
